# curdate 命令行工具设计

## 项目概述

创建一个 Python 命令行工具 `curdate`，在终端清晰展示当前日期时间、农历、法定节假日和当月日历。

## 功能规格

### 1. 日期时间显示
- 格式：`2026年03月17日 21:30:45`
- 颜色：年月日（白色）、时分秒（浅蓝色）

### 2. 星期显示
- 格式：`星期二`
- 颜色：浅绿色

### 3. 农历显示
- 格式：`农历二月十八`
- 颜色：黄色

### 4. 法定节假日
- 显示当前日期是否为节假日，若是则用红色突出显示节日名称
- 显示下一个法定节假日名称、日期及距离天数
- 颜色：节假日（红色）、调休（橙色）、下一个假期（紫色）

### 5. ASCII 日历
- 类似 `cal` 命令效果
- 显示当月日历，星期作为表头
- 今日日期用反色或下划线标注

## 技术方案

### 依赖库
- `zhdate` - 农历转换
- `holidays` - 中国法定节假日
- `termcolor` - 终端着色

### 安装
```bash
pip install zhdate holidays termcolor
```

## 设计原则

- 单一命令，简单直接
- 带颜色输出，重点信息突出
- 仅显示当月日历
